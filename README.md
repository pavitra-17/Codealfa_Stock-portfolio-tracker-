# Codealfa_Stock-portfolio-tracker-# CodeAlpha_StockPortfolioTracker

## 📌 Project Description
This project is a simple stock portfolio tracker created using Python.  
Users can enter stock names and quantities to calculate the total investment value.

## 🚀 Features
- Calculates stock investment value
- Uses predefined stock prices
- Saves portfolio details in a text file
- Easy console interaction

## 🛠 Technologies Used
- Python

## ▶️ How to Run
1. Install Python
2. Open terminal or command prompt
3. Run the file using:

python stock_tracker.py

## 📚 Concepts Used
- Dictionaries
- Loops
- File Handling
- User Input
- Arithmetic Operations
